Feature Ideas

The project will include several different technologies and the basic idea is to create an automatic content reputation system which rates web page or file content.

This list contains an overview of ideas that will be implemented/performed:

  • Connect Python to Postgres
  • Unit testing different technologies
  • Test automation
  • Using Selenium with Python to test the front end
  • Writing content parsers in Python for web pages and files
  • Caching content with Redis
  • Real time view of content ratings
  • Deployment automation
  • Split the crawling task into multiple servers
  • Increase load and improve performance by optimzing different components
  • Writeing blog posts explaining different techniques used in this sample project
